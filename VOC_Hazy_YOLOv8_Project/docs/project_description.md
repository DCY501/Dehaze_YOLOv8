# 课程设计说明

## 题目

基于 YOLOv8 的雾天图像目标检测实验。

## 项目目标

本项目围绕 VOC_hazy 数据集开展目标检测实验，主要流程包括：

1. 使用 `tools/prepare_voc_hazy.py` 下载 VOC2007，并按大气散射模型生成带雾图像。
2. 使用 `configs/VOC_hazy.yaml` 配置 YOLO 数据集路径和 20 个 VOC 类别。
3. 使用 YOLOv8 进行雾天目标检测训练和预测。
4. 保留 `configs/yolov8-dehaze.yaml` 作为后续多任务/去雾辅助结构实验的模型配置入口。

## 不上传的内容

为了避免仓库过大，本项目没有上传完整数据集、训练输出和模型权重文件，包括：

- `datasets/`
- `runs/`
- `*.pt`
- 压缩包和缓存文件

这些文件可按 README 中的命令重新生成或另行提供。
